
# Supporting Information

This repository contains (1) Relaxed structures (VASP POSCAR format) of M11−xM2xN alloys, where M1 = Al, Ga, In and M2 = Sc, La, Y, B. Each alloy composition (x) is represented by four distinct structural models (_v0, _v1, _v2, _v3). (2) [Band gaps of M11−xM2xN alloys](https://github.com/prashungorai/papers/blob/45a59b0d0ea8ef8abbbfd97d7caf025eed5ace48/2026/heterostructure-bandgap/HSE_band_gap_all_alloys.csv) calculated using hybrid DFT functional, (3) Summary of bond lengths, bond angles, and cation coordination for each alloy, and (3) Python scripts for calculating the bond lengths, bond angles, and cation coordination.

This repository accompanies the paper "Rethinking How Band Gap Changes Enable Ferroelectricity in Wurtzite Nitride Alloys".
